Luis Carrillo - Blackjack Term Project - readme

- Run the program and enter any appropriate input in the program running in Python's immediate mode
- If prompted for a wager, input any number from 25 (the minimum wager) to the total amount of dollars you have at that point in the game (you start with 500).
- If prompted to choose between hitting or standing, input 'H' or 'h' to hit and 'S' or 's' to stand.
- If asked whether or not you want to play another round, input 'Y' or 'y' for Yes and 'N' or 'n' for No.
- If prompted for your initials, enter your initials and you will be shown the post-game screen. The program stops running once the post-game screen is displayed.
